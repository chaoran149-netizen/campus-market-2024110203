# 错题：级数比较判别法

## 题目
判断级数是否收敛，并说明使用判别法的条件。

## 我的错误答案
直接使用比较判别法，没有先说明正项条件。

## 正确答案
先确认通项非负，再选择合适的比较对象，并说明比较关系在充分大 n 后成立。

## 错因分类
- [ ] 概念不清
- [x] 条件遗漏
- [ ] 公式记错
- [ ] 计算错误
- [ ] 方法选择错误
- [ ] 表达不规范

## 最小错误点
没有先判断是否为正项级数，直接使用比较判别法。

## 修复说明
比较判别法不是只看大小关系，还需要说明通项符号和比较对象的收敛性/发散性。

## 变式题
1. 基础题：判断一个正项级数能否直接比较。
2. 变式题：给出非正项通项，说明为什么不能直接使用比较判别法。
3. 综合题：在比较判别法和极限比较判别法之间选择。
